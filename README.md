![메인 화면](/suseok/images/Home.PNG)
메인 화면에서는 사용자가 로그인을 할 수 있으며, 기본 기능들에 접근할 수 있는 Nav Bar와 주요 기능들에 접근할 수 있는 HanBurger Button이 있습니다.

## 로그인 화면
![로그인 화면](path/to/your/login_screen_image.png)

## 게시판 화면
